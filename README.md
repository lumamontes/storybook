### :wrench:  Installation

1. Clone
```sh 
git@github.com:lumamontes/spacetravelingblog.git
```
2. Install packages
```sh 
yarn 
``` 
4. Start server
```sh 
yarn start
```
5. Running at http://localhost:3000/
